| 항목 | 캠퍼스 Top1 | 주말 데일리 Top1 |

| TPO 적합성 | ✅ 캠퍼스에 적합 | ✅ 주말 외출에 적합 |
| 연령 적합성 | ✅ 캐주얼, 젊은 느낌 | ⚠️ 니트 가디건이 30대 느낌 |
| 가격 적절성 | ✅ ₩52K — 대학생 가성비 OK | ⚠️ ₩97K — 대학생에게 부담 |
| 스타일 자연스러움 | ⚠️ 후드+청바지 반복 | ✅ 자연스러움 |

**핵심 발견:**
- 캠퍼스 ₩52,500은 적절. 하지만 **8건 중 전부 후드+청바지** — 조합 다양성 0
- 주말 ₩97,210은 대학생에게 고가. 예산 3~10만원을 설정했는데 상한에 가까움
- 상품명 "H&M 리본 디테일 니트 가디건" — 20대 초반보다 20대 후반~30대 느낌

|------|---------|-----------|
| 코디 | 니트 + 슬랙스 ₩91,700 | 니트 + 슬랙스 + 힐 ₩98,040 |
| TPO 적합성 | ✅ 출근에 적합 | ✅ 데이트에 적합 |
| 연령 적합성 | ✅ 사회초년생에 적합 | ✅ 자연스러움 |
| 가격 적절성 | ✅ ₩91K — 적절 | ✅ ₩98K — 적절 |
| 스타일 자연스러움 | ✅ | ✅ |
| 구매 의사 | **Yes** | **Yes** |
| 실패 원인 | 없음 | 없음 |

**핵심 발견:**
- **가장 잘 맞는 연령대.** ColorFit의 현재 데이터가 20대 후반 여성에 최적화
- core "프로페셔널한 출근 조합" + evidence "깔끔한 실루엣이 전문적인 분위기" — 적절
- 단, 출근 Top1~3이 전부 니트+슬랙스 — **다양성 부족**

| 항목 | 면접 Top1 | 비즈니스 출근 Top1 |
|------|---------|-----------------|

| TPO 적합성 | ✅ 면접에 적합 | ✅ 출근에 적합 |
| 연령 적합성 | ⚠️ 상품명 "안다르 에어로코튼" — 요가/운동복 브랜드 | ✅ |

| 스타일 자연스러움 | ⚠️ "안다르" 셔츠가 면접 포멀과 불일치 | ✅ |
| 구매 의사 | **No** (브랜드 신뢰 부족) | **Yes** |

- **브랜드 기반 필터가 필요** — 운동복 브랜드(안다르, 젝시믹스 등)는 면접/출근에서 제외

## 3. 연령별 실패 패턴 분석

| 연령대 | 주요 실패 패턴 | 근본 원인 |
|--------|-------------|----------|
| **20대 초반** | 가격 높음 + 조합 단조 (후드+청바지 반복) | 연령별 가격 가중치 없음, 조합 다양성 부족 |
| **20대 후반** | 가장 적합 — 큰 문제 없음 | 현재 데이터가 이 연령대 최적화 |
| **30대** | 운동복 브랜드(안다르)가 면접에 노출 | 브랜드 등급/카테고리 크로스체크 없음 |
| **40대** | 스포츠 브랜드(아디다스, 내셔널지오)가 행사에 노출, 품격 부족 | 브랜드 포멀도 필터 없음, 연령 적합성 필터 없음 |
| **선물용** | 사이즈 민감 아이템 포함 | 구매 목적 필터 없음 |
| **공통** | 침대 커버가 스커트로 분류 | 카테고리 분류 오류 잔존 |

## 4. 누락된 필터 정의

### 현재 존재하는 필터
- ✅ 성별 (H1)
- ✅ 예산 (H2)
- ✅ 계절 (H3)
- ✅ TPO (H4/Eligibility)
- ✅ 브랜드 화이트리스트 (H5)
- ✅ 톤 (H7)
- ✅ 스타일 (H8)
- ✅ 스타일리스트 룰 (카테고리 금기/포멀도)
- ✅ QA Gate (성별/시즌/포멀도)

### 누락된 필터 (신규 필요)

| # | 필터 | 설명 | 영향 범위 | 구현 난이도 |
|---|------|------|----------|-----------|
| **F1** | 브랜드 등급 필터 | 스포츠/아웃도어 브랜드를 면접/행사에서 제외 | 면접, 행사, 출근 | 중 (브랜드 분류 dict 필요) |
| **F2** | 상품명 타겟 연령 필터 | "엄마", "할머니", "중년" 키워드 포함 시 20대에서 제외 | 전 TPO | 하 (키워드 블랙리스트) |

## 5. 개선 우선순위 Top 5

| 순위 | 개선 | 영향 | 즉시 가능 여부 |
|------|------|------|-------------|
| **1** | 비의류 아이템 제거 ("침대스커트" 등) | 치명적 신뢰 파괴 방지 | ✅ 키워드 블랙리스트 추가 |
| **2** | 브랜드 등급 필터 (스포츠 브랜드 → 면접/행사 제외) | 30~40대 신뢰도 | ✅ 브랜드 분류 dict |
| **3** | 상품명 연령 키워드 필터 ("엄마"/"할머니"/"중년" 제외) | 20대 사용자 경험 | ✅ 키워드 블랙리스트 |
| **4** | 코디 조합 다양성 강화 (후드+청바지 반복 방지) | 전 연령 체감 | ⚠️ 리랭킹 로직 개선 필요 |
| **5** | 온보딩에 연령대 수집 추가 (향후) | 전체 개인화 | ❌ 온보딩 수정 필요 |

### 한 줄 결론

> **ColorFit은 20대 후반 여성에게 가장 잘 맞는다. 20대 초반(가격), 30대(브랜드 신뢰), 40대(품격)에서 실패하며, 근본 원인은 "연령 필터 부재"와 "브랜드 등급 분류 부재"다. 비의류 데이터 오류("침대스커트")는 즉시 수정 필요.**
