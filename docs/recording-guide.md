# ColorFit 데모 영상 녹화 가이드

## 녹화 환경 설정

### 도구
- **화면 녹화:** OBS Studio 또는 Windows Game Bar (Win+G)
- **해상도:** 1080x1920 (모바일 세로) 또는 1920x1080 (가로 + DevTools)
- **프레임:** 30fps
- **포맷:** MP4 (H.264)

### Chrome DevTools 모바일 모드
1. F12 → 디바이스 툴바 (Ctrl+Shift+M)
2. 기기: **iPhone 14 Pro (390x844)**
3. 확대: 75~100% (화면에 맞게 조절)

### 사전 준비
- [ ] 백엔드 서버 실행 (`uvicorn app.main:app --reload --port 8000`)
- [ ] 프론트엔드 실행 (`npm run dev`)
- [ ] DB에 테스트 데이터 존재 확인 (코디 20개+, 상품 데이터)
- [ ] localStorage 초기화 (시나리오 A는 첫 방문)
- [ ] 브라우저 북마크바/확장 프로그램 숨기기
- [ ] 알림/팝업 모두 비활성화

## 녹화 팁

1. **마우스 커서를 천천히** 움직이기 — 시청자가 따라갈 수 있도록
2. **탭 후 1~2초 멈춤** — 화면 전환 후 내용 확인 시간
3. **스크롤은 부드럽게** — 급하지 않게
4. **실수 시 처음부터** 다시 녹화 (편집 최소화)
5. 시나리오 A와 D를 **별도로 녹화**해도 OK (나중에 이어붙이기)

## 편집 시 추가 요소 (선택)

- 배경 음악: 밝고 가벼운 루프 BGM (저작권 프리)
- 클릭 위치 강조: 마우스 하이라이트 효과
- 자막: 핵심 포인트마다 하단 자막 추가
- 속도: 대기 시간이 길면 1.5배속 편집
